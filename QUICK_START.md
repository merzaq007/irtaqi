# 🚀 دليل البدء السريع - منصة ارتقي

## ⚡ البدء في 5 دقائق

### 1️⃣ التثبيت
```bash
# استنساخ المشروع
git clone <repository-url>
cd irtaqi_platform

# تثبيت الحزم
npm install
```

### 2️⃣ التشغيل
```bash
# تشغيل المشروع
npm run dev

# افتح المتصفح على
# http://localhost:5173
```

### 3️⃣ الاستكشاف
- **الصفحة الرئيسية**: `/`
- **صفحة مقياس**: `/module/web-apps`
- **لوحة التحكم**: `/admin` (كلمة المرور: `admin123`)
