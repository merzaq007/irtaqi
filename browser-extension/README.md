# 🚀 Irtaqi Moodle Sync - دليل التثبيت والاستخدام

## ✨ المزايا

✅ **مزامنة تلقائية** - يكتشف الملفات الجديدة في المودل تلقائياً  
✅ **إشعارات فورية** - يخبرك عند نشر محاضرة جديدة  
✅ **رفع تلقائي** - يرفع الملفات لمنصة ارتقي بدون تدخل يدوي  
✅ **زر سريع** - زر مزامنة سريع في صفحات المودل  
✅ **يعمل بدون API** - لا يحتاج Web Services من المودل

## 📦 التثبيت

### الخطوة 1: تحضير الإضافة

1. **افتح مجلد المشروع:**
   ```
   C:\Users\billionaire2025\Desktop\irtaqi_academic_platform\browser-extension
   ```

2. **تأكد من وجود الملفات:**
   - `manifest.json`
   - `content.js`
   - `background.js`
   - `popup.html`
   - `popup.js`

### الخطوة 2: إنشاء الأيقونات

قم بإنشاء 3 صور للأيقونة (أو استخدم أي صورة مؤقتة):
- `icon16.png` (16x16 بكسل)
- `icon48.png` (48x48 بكسل)
- `icon128.png` (128x128 بكسل)

**أو استخدم هذا الأمر لإنشاء أيقونات بسيطة:**

---

### الخطوة 3: تثبيت الإضافة في Chrome/Edge

1. **افتح المتصفح (Chrome أو Edge)**

2. **اذهب إلى صفحة الإضافات:**
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

3. **فعّل "وضع المطور" (Developer mode)**
   - ستجده في الزاوية العلوية اليمنى

4. **اضغط "Load unpacked" أو "تحميل إضافة غير معبأة"**

5. **اختر مجلد:**
   ```
   C:\Users\billionaire2025\Desktop\irtaqi_academic_platform\browser-extension
   ```

6. **ستظهر الإضافة في قائمة الإضافات! ✅**

## 🛠️ استكشاف الأخطاء

### المشكلة: لا تظهر إشعارات

**الحل:**
- تأكد من تفعيل الإشعارات في إعدادات المتصفح
- أعد تحميل صفحة المودل

### المشكلة: لا يتم رفع الملفات

**الحل:**
- تأكد من إدخال Supabase Key الصحيح
- افتح Console (F12) وابحث عن أخطاء

### المشكلة: الزر لا يظهر في المودل

**الحل:**
- أعد تحميل الصفحة (F5)
- تأكد من تثبيت الإضافة بشكل صحيح
